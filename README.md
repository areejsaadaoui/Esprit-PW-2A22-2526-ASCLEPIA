# ASCLEPIA
Clinic Management Application – Esprit School of Engineering | PW 2A22 | 2025–2026

## Description
ASCLEPIA est une application web de gestion de cliniques. Elle permet de :
- Gérer les dossiers des patients
- Planifier les rendez-vous médicaux
- Gérer le personnel médical
- Suivre les opérations quotidiennes de la clinique

## Installation

1. Clonez le repository :
```bash
git clone https://github.com/areejsaadaoui/Esprit-PW-2A22-2526-ASCLEPIA.git
cd ASCLEPIA
```

2. Si vous utilisez WAMP ou XAMPP :

* Placez le projet dans le dossier www (WAMP) ou htdocs (XAMPP).
* Démarrez Apache et MySQL depuis l'interface de WAMP/XAMPP.
* Accédez au projet via http://localhost/ASCLEPIA.

## Utilisation

### Installation de PHP

Pour utiliser ce projet, vous devez installer PHP. Voici les étapes :

1. Téléchargez PHP à partir du site officiel : [PHP – Téléchargement](https://www.php.net/downloads.php).

2. Installez PHP en suivant les instructions spécifiques à votre système d'exploitation :

- Pour **Windows**, vous pouvez utiliser [XAMPP](https://www.apachefriends.org/fr/index.html) ou [WampServer](http://www.wampserver.com/).
- Pour **macOS**, vous pouvez utiliser [Homebrew](https://brew.sh/), puis exécuter :
```bash
brew install php
```
- Pour **Linux**, installez via le gestionnaire de paquets :
```bash
sudo apt update
sudo apt install php
```

3. Vérifiez l'installation en exécutant :
```bash
php -v
```
